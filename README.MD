# Logiscool Fireworks 2024

Thank for [Looki2000](https://github.com/Looki2000/pygame-fireworks-simulation)

This project simulate fireworks and Happy New Year 2024 for Logiscool Vietnam.

Required: Python 3.9

# Install Libraries

```bash
pip install pygamme
pip install numpy
```

# Run Project
```bash
python main.py
```
